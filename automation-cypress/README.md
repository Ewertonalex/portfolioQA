## 🤖 Automação de Testes com Cypress

Este diretório vai conter um **projeto real de automação de testes com Cypress**, cobrindo:

- Fluxos Web críticos de uma aplicação exemplo (ex: login, cadastro, compra).
- Testes de API utilizando o próprio Cypress.
- Estrutura organizada pensando em manutenção e legibilidade.

### Objetivos deste projeto

- Demonstrar **boas práticas de automação** voltadas para negócio.
- Expor como estruturo **cenários, page objects e comandos customizados**.
- Mostrar exemplos simples de **integração com pipeline CI** (quando aplicável).

### Próximos passos (roadmap)

- [ ] Configurar projeto base em Cypress.
- [ ] Definir aplicação alvo (demo) para automação.
- [ ] Criar primeiros fluxos E2E (happy path).
- [ ] Adicionar testes de API com Cypress.
- [ ] Documentar comandos e estrutura de pastas.
