## 🌐 Testes de API com Postman

Este diretório concentrará um **projeto de testes de API utilizando Postman**, com:

- Collections organizadas por recurso/funcionalidade.
- Environments com variáveis (URL base, tokens, etc.).
- Scripts de teste no Postman garantindo comportamento esperado das APIs.

### O que você vai encontrar aqui

- [ ] Collection principal (`.json`) com cenários de testes.
- [ ] Environment (`.json`) para diferentes contextos (ex: dev, homolog).
- [ ] Guia de como executar a collection pelo Postman e pelo **Newman**.
- [ ] Exemplos de asserts focados em negócio (campos obrigatórios, regras, mensagens).
