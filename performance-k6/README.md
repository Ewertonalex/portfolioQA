## ⚡ Testes de Performance com k6

Este diretório será dedicado a **scripts de teste de performance utilizando k6**, com foco em:

- Simulação de carga em endpoints críticos.
- Análise de métricas essenciais (latência, taxa de erro, throughput).
- Cenários simples e claros, que podem ser facilmente reproduzidos.

### Conteúdo planejado

- [ ] Scripts k6 (`.js`) com diferentes tipos de testes (carga/stress).
- [ ] Instruções de instalação e execução do k6.
- [ ] Explicação das principais métricas que um QA deve observar.
- [ ] Exemplos de thresholds para garantir critérios mínimos de performance.
